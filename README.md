### 前言

​	该项目为北京信息科技大学软件工程系大三下学期的软件开发实践的期末大作业，具体项目要求详见根目录下的`Spring Boot实验报告参考模版-2021 (2).doc`文档。本次实验，我们采用的分工为前端+后端的形式进行对接。
